## Getting Started with Tailwind

Setting up Tailwind CSS in your project is straightforward:

```bash
# Install Tailwind CSS
npm install -D tailwindcss postcss autoprefixer

# Generate configuration files
npx tailwindcss init -p
```

Next, configure your template paths in `tailwind.config.js`:

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./pages/**/*.{js,ts,jsx,tsx}",
    "./components/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

<TipCard title="Pro Tip">
  Tailwind uses a mobile-first approach. This means styles with no prefix apply to all screen sizes, while prefixed variants only apply at that breakpoint and above.
</TipCard>
